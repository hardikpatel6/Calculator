# Simple Web Calculator

A basic calculator implemented using HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - The main HTML file containing the calculator layout
- `index.css` - Stylesheet for the calculator design
- `index.js` - JavaScript code for calculator functionality

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean and responsive interface
- Browser-based calculator

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start calculating!

## Development

To modify this project:
- Edit `index.html` to change the calculator layout
- Modify `index.css` to adjust the styling
- Update `index.js` to change the calculator logic
